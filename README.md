# Sac Pathways (static website)

## What’s included
- `index.html` — the page
- `styles.css` — styling (navy + gold theme)
- `script.js` — mobile menu + demo contact form
- `assets/hero-bg.jpg` — background image (from your screenshot)
- `assets/section-bg.png` — secondary background image

## Run locally
Option A (double-click):
- Open `index.html` in your browser.

Option B (recommended: local server):
- VS Code: install "Live Server" and click "Go Live"
- Or run:
  - Python 3: `python -m http.server 8000`
  - Then open: http://localhost:8000

## Customize
- Colors: edit CSS variables in `styles.css` under `:root`
- Hero text/buttons: edit the Hero section in `index.html`
- Replace images: drop new files into `assets/` and update URLs in `styles.css`
